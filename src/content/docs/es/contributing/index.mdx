OpenCV se mantiene en GitHub en `github.com/opencv/opencv` y acepta contribuciones a través de pull requests. El repositorio `opencv_contrib` (`github.com/opencv/opencv_contrib`) contiene módulos adicionales y experimentales — los nuevos algoritmos generalmente comienzan allí antes de ser promovidos al repositorio principal. El `CONTRIBUTING.md` oficial del repositorio principal es un redireccionamiento a la wiki de GitHub en `github.com/opencv/opencv/wiki/How_to_contribute`.

<Aside type="note">
OpenCV utiliza la licencia **Apache 2.0** (migrada desde BSD de 3 cláusulas en la versión 4.5.0). Todas las contribuciones deben ser compatibles con esta licencia. No existe un CLA formal — los colaboradores aceptan la licencia mediante la casilla de verificación de la plantilla de PR.
</Aside>

## Los Tres Repositorios que Necesitas

La mayoría de los colaboradores trabajan con tres repositorios:

| Repositorio | Propósito |
|---|---|
| `github.com/opencv/opencv` | Biblioteca principal — el repositorio central |
| `github.com/opencv/opencv_contrib` | Módulos adicionales — los nuevos algoritmos comienzan aquí |
| `github.com/opencv/opencv_extra` | Datos de prueba — necesarios para ejecutar la suite de pruebas |

Cuando añadas un nuevo algoritmo, debe enviarse primero a `opencv_contrib`. Tras demostrar estabilidad e interés de la comunidad, puede ser promovido al repositorio principal.

<Aside type="tip">
Si tienes una pregunta sobre el **uso** de OpenCV, NO abras un issue en GitHub. Usa el foro de preguntas y respuestas de OpenCV en `forum.opencv.org` o Stack Overflow con la etiqueta `opencv`. Los issues de GitHub están reservados para bugs confirmados y propuestas de características.
</Aside>
