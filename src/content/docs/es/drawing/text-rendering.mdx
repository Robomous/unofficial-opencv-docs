Agregar texto a imágenes es una de las tareas más comunes en visión por computadora: etiquetar objetos detectados, mostrar contadores de FPS, estampar marcas de tiempo o mostrar resultados de clasificación. OpenCV proporciona `cv2.putText()` para renderizar texto y `cv2.getTextSize()` para medir las dimensiones del texto antes de dibujarlo.

## `cv2.putText()` — Renderizar Texto

La firma completa para renderizar texto sobre una imagen es:

```python
cv2.putText(img, text, org, fontFace, fontScale, color, thickness=1, lineType=cv2.LINE_8, bottomLeftOrigin=False)
```

- **`img`** — Imagen sobre la que se dibuja (se modifica en su lugar)
- **`text`** — La cadena de texto a renderizar
- **`org`** — Esquina inferior izquierda de la cadena de texto `(x, y)`
- **`fontFace`** — Constante del tipo de fuente (ej. `cv2.FONT_HERSHEY_SIMPLEX`)
- **`fontScale`** — Multiplicador del tamaño de fuente relativo al tamaño base
- **`color`** — Color del texto como `(B, G, R)`
- **`thickness`** — Grosor del trazo en píxeles (por defecto 1)
- **`lineType`** — Usa `cv2.LINE_AA` para texto con anti-aliasing (suavizado)
- **`bottomLeftOrigin`** — Si es `True`, el origen está en la esquina inferior izquierda de la imagen (rara vez se usa)

```python
import cv2
import numpy as np

canvas = np.zeros((300, 600, 3), dtype=np.uint8)

cv2.putText(canvas, "Hello OpenCV", (50, 150),
            cv2.FONT_HERSHEY_SIMPLEX, 1.5, (255, 255, 255), 2, cv2.LINE_AA)

cv2.imshow("Text", canvas)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

<Aside type="caution">
El parámetro `org` especifica la **esquina inferior izquierda de la línea base del texto**, no la esquina superior izquierda. Esto significa que la coordenada y marca donde se asienta la parte inferior de letras como "a" y "o" — los descendentes como "g" e "y" se extienden por debajo de este punto. Si necesitas colocar texto en una posición superior izquierda específica, primero suma la altura del texto a la coordenada y.
</Aside>

## Tipos de Fuente

OpenCV incluye las fuentes Hershey. Cada una tiene un estilo y peso distintos:

- **`FONT_HERSHEY_SIMPLEX`** — Sans-serif de tamaño normal
- **`FONT_HERSHEY_PLAIN`** — Sans-serif de tamaño pequeño
- **`FONT_HERSHEY_DUPLEX`** — Sans-serif, más gruesa que SIMPLEX
- **`FONT_HERSHEY_COMPLEX`** — Serif de tamaño normal
- **`FONT_HERSHEY_TRIPLEX`** — Serif, más gruesa que COMPLEX
- **`FONT_HERSHEY_COMPLEX_SMALL`** — Versión más pequeña de COMPLEX
- **`FONT_HERSHEY_SCRIPT_SIMPLEX`** — Estilo manuscrito
- **`FONT_HERSHEY_SCRIPT_COMPLEX`** — Manuscrita, trazos más gruesos
- **`FONT_ITALIC`** — Flag que se combina con cualquier fuente mediante OR a nivel de bits

El siguiente código renderiza cada tipo de fuente en un solo lienzo:

```python
import cv2
import numpy as np

fonts = [
    (cv2.FONT_HERSHEY_SIMPLEX, "SIMPLEX"),
    (cv2.FONT_HERSHEY_PLAIN, "PLAIN"),
    (cv2.FONT_HERSHEY_DUPLEX, "DUPLEX"),
    (cv2.FONT_HERSHEY_COMPLEX, "COMPLEX"),
    (cv2.FONT_HERSHEY_TRIPLEX, "TRIPLEX"),
    (cv2.FONT_HERSHEY_COMPLEX_SMALL, "COMPLEX_SMALL"),
    (cv2.FONT_HERSHEY_SCRIPT_SIMPLEX, "SCRIPT_SIMPLEX"),
    (cv2.FONT_HERSHEY_SCRIPT_COMPLEX, "SCRIPT_COMPLEX"),
    (cv2.FONT_HERSHEY_SIMPLEX | cv2.FONT_ITALIC, "SIMPLEX + ITALIC"),
]

canvas = np.zeros((450, 700, 3), dtype=np.uint8)

for i, (font, name) in enumerate(fonts):
    y = 40 + i * 45
    cv2.putText(canvas, name, (20, y), font, 1.0, (0, 255, 200), 1, cv2.LINE_AA)

cv2.imshow("Font Faces", canvas)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

## Escala de Fuente y Grosor

`fontScale` es un multiplicador sobre el tamaño base de la fuente: `0.5` para la mitad, `1.0` para el valor por defecto, `2.0` para el doble. `thickness` controla el grosor del trazo. Siempre usa `cv2.LINE_AA` para bordes suavizados con anti-aliasing.

```python
import cv2
import numpy as np

canvas = np.zeros((350, 800, 3), dtype=np.uint8)

# Diferentes escalas de fuente
cv2.putText(canvas, "Scale 0.5", (30, 40), cv2.FONT_HERSHEY_SIMPLEX, 0.5, (200, 200, 200), 1, cv2.LINE_AA)
cv2.putText(canvas, "Scale 1.0", (30, 100), cv2.FONT_HERSHEY_SIMPLEX, 1.0, (200, 200, 200), 1, cv2.LINE_AA)
cv2.putText(canvas, "Scale 2.0", (30, 200), cv2.FONT_HERSHEY_SIMPLEX, 2.0, (200, 200, 200), 1, cv2.LINE_AA)

# Diferentes grosores
cv2.putText(canvas, "Thin (1)", (450, 80), cv2.FONT_HERSHEY_SIMPLEX, 1.0, (100, 255, 100), 1, cv2.LINE_AA)
cv2.putText(canvas, "Medium (2)", (450, 160), cv2.FONT_HERSHEY_SIMPLEX, 1.0, (100, 255, 100), 2, cv2.LINE_AA)
cv2.putText(canvas, "Bold (4)", (450, 240), cv2.FONT_HERSHEY_SIMPLEX, 1.0, (100, 255, 100), 4, cv2.LINE_AA)

cv2.imshow("Scale and Thickness", canvas)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

## Medir Texto — `cv2.getTextSize()`

Antes de colocar texto, a menudo necesitas saber cuánto espacio ocupará. `cv2.getTextSize()` devuelve las dimensiones sin dibujar:

```python
(width, height), baseline = cv2.getTextSize(text, fontFace, fontScale, thickness)
```

Devuelve `((width, height), baseline)`. El `baseline` es la distancia debajo de la parte inferior del texto hasta el punto más bajo de los descendentes como "g" y "p".

### Centrar Texto

```python
import cv2
import numpy as np

canvas = np.zeros((400, 600, 3), dtype=np.uint8)
text = "Centered Text"
font = cv2.FONT_HERSHEY_SIMPLEX
scale = 1.5
thickness = 2

(text_w, text_h), baseline = cv2.getTextSize(text, font, scale, thickness)

# Calcular la posición para centrar el texto
x = (canvas.shape[1] - text_w) // 2
y = (canvas.shape[0] + text_h) // 2

cv2.putText(canvas, text, (x, y), font, scale, (255, 255, 255), thickness, cv2.LINE_AA)

cv2.imshow("Centered", canvas)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

### Dibujar un Rectángulo de Fondo

Dibuja un rectángulo relleno detrás del texto para mejorar la legibilidad — el mismo patrón utilizado para las cajas de etiquetas de detección:

```python
import cv2
import numpy as np

canvas = np.zeros((300, 500, 3), dtype=np.uint8)
text = "Label Box"
font = cv2.FONT_HERSHEY_SIMPLEX
scale = 1.0
thickness = 2
padding = 10

(text_w, text_h), baseline = cv2.getTextSize(text, font, scale, thickness)

# Esquina superior izquierda de la etiqueta
org_x, org_y = 50, 100

# Rectángulo de fondo: considera la altura del texto, línea base y relleno
rect_top_left = (org_x - padding, org_y - text_h - padding)
rect_bottom_right = (org_x + text_w + padding, org_y + baseline + padding)

cv2.rectangle(canvas, rect_top_left, rect_bottom_right, (0, 120, 255), cv2.FILLED)
cv2.putText(canvas, text, (org_x, org_y), font, scale, (255, 255, 255), thickness, cv2.LINE_AA)

cv2.imshow("Label Box", canvas)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

## Texto Multilínea

OpenCV **no** soporta `\n` en `cv2.putText()` — los saltos de línea se ignoran o se renderizan como caracteres literales. La solución es dividir el texto en líneas, calcular la altura de línea con `getTextSize()`, y renderizar cada línea en una posición y incrementada:

```python
import cv2
import numpy as np


def put_multiline_text(img, text, org, font, scale, color, thickness, line_spacing=1.5):
    """Renderiza texto multilínea. org es la esquina superior izquierda del bloque de texto."""
    x, y = org
    lines = text.split("\n")

    for i, line in enumerate(lines):
        (_, line_h), _ = cv2.getTextSize(line, font, scale, thickness)
        line_y = y + int(i * line_h * line_spacing) + line_h
        cv2.putText(img, line, (x, line_y), font, scale, color, thickness, cv2.LINE_AA)


canvas = np.zeros((400, 600, 3), dtype=np.uint8)

message = "Line 1: Hello\nLine 2: OpenCV\nLine 3: Multiline\nLine 4: Text!"
put_multiline_text(canvas, message, (30, 30),
                   cv2.FONT_HERSHEY_SIMPLEX, 1.0, (255, 255, 255), 2)

cv2.imshow("Multiline", canvas)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

<Aside type="note">
Las fuentes integradas de OpenCV están limitadas a caracteres ASCII básicos. Si necesitas texto Unicode, emojis o fuentes TrueType, usa **PIL/Pillow** para renderizar sobre un array de NumPy:

```python
from PIL import Image, ImageDraw, ImageFont
pil_img = Image.fromarray(cv2.cvtColor(img, cv2.COLOR_BGR2RGB))
draw = ImageDraw.Draw(pil_img)
draw.text((x, y), "Unicode: ❤", font=ImageFont.truetype("arial.ttf", 32))
img = cv2.cvtColor(np.array(pil_img), cv2.COLOR_RGB2BGR)
```
</Aside>

## Ejemplo Práctico — Superposición de Contador FPS

Las aplicaciones de visión en tiempo real típicamente muestran un contador de FPS en una esquina. Un fondo semitransparente mantiene el texto legible sobre cualquier escena:

```python
import cv2
import numpy as np

def draw_fps(frame, fps, position="top-left"):
    """Dibuja un contador de FPS con fondo semitransparente."""
    text = f"FPS: {fps:.1f}"
    font = cv2.FONT_HERSHEY_SIMPLEX
    scale, thickness, padding = 0.7, 2, 8

    (text_w, text_h), baseline = cv2.getTextSize(text, font, scale, thickness)
    x = frame.shape[1] - text_w - 2 * padding if position == "top-right" else 0
    bg_w, bg_h = text_w + 2 * padding, text_h + baseline + 2 * padding

    # Superposición semitransparente
    overlay = frame.copy()
    cv2.rectangle(overlay, (x, 0), (x + bg_w, bg_h), (0, 0, 0), cv2.FILLED)
    cv2.addWeighted(overlay, 0.6, frame, 0.4, 0, frame)

    # Dibujar texto encima
    cv2.putText(frame, text, (x + padding, padding + text_h),
                font, scale, (0, 255, 0), thickness, cv2.LINE_AA)
    return frame

# Demostración
canvas = np.zeros((480, 640, 3), dtype=np.uint8)
canvas[:] = (60, 60, 60)
draw_fps(canvas, 29.97, position="top-left")

cv2.imshow("FPS Overlay", canvas)
cv2.waitKey(0)
cv2.destroyAllWindows()
```

## Ejemplo Práctico — Etiquetas de Detección

Toda visualización de detección de objetos sigue el mismo patrón: un cuadro delimitador con una barra de etiqueta rellena encima que muestra el nombre de la clase y la confianza. Aquí tienes una función reutilizable para esto:

```python
import cv2
import numpy as np


def draw_detection(img, bbox, label, confidence, color=(0, 255, 0)):
    """Dibuja un cuadro delimitador con una barra de etiqueta rellena encima."""
    x1, y1, x2, y2 = bbox
    font = cv2.FONT_HERSHEY_SIMPLEX
    scale = 0.6
    thickness = 1

    # Dibujar cuadro delimitador
    cv2.rectangle(img, (x1, y1), (x2, y2), color, 2)

    # Construir texto de la etiqueta
    text = f"{label} {confidence:.2f}"
    (text_w, text_h), baseline = cv2.getTextSize(text, font, scale, thickness)

    # Dibujar rectángulo relleno encima del cuadro delimitador para la etiqueta
    label_y1 = max(y1 - text_h - baseline - 6, 0)
    label_y2 = y1
    cv2.rectangle(img, (x1, label_y1), (x1 + text_w + 8, label_y2), color, cv2.FILLED)

    # Dibujar texto de la etiqueta (negro sobre fondo de color)
    cv2.putText(img, text, (x1 + 4, y1 - baseline - 3),
                font, scale, (0, 0, 0), thickness, cv2.LINE_AA)


# Demostración: simular resultados de detección en un lienzo vacío
canvas = np.zeros((500, 700, 3), dtype=np.uint8)
canvas[:] = (40, 40, 40)

draw_detection(canvas, (50, 80, 250, 350), "person", 0.95, (0, 255, 0))
draw_detection(canvas, (300, 120, 480, 400), "dog", 0.87, (255, 150, 0))
draw_detection(canvas, (500, 60, 670, 220), "car", 0.72, (0, 100, 255))

cv2.imshow("Detection Labels", canvas)
cv2.waitKey(0)
cv2.destroyAllWindows()
```
